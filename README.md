## General info

Project contains four pages and navigation bar, each page displays informations about me:
Header - you can navigate through entire project
About - there are some informations about me and my hobbys
Technologies - this page contains all technologies I have learned
Projects - there is slider which shows my projects
Contact - page for contact with me
	
## Technologies
Project is created with:
* JavaScript
* React
* Tailwind CSS
* Flowbite
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../portfolio
$ npm install
$ npm start
```
or visit it https://portfolio-jakubhajduk53.vercel.app/
